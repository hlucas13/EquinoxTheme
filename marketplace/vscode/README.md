# Equinox Colors

A carefully crafted color theme for developers who spend long hours in their editor. Equinox Colors is built around two principles: **ergonomic contrast** and **visual calm** ‚Äî every color pair meets **WCAG AAA (7:1)** contrast, and the palette avoids harsh saturation that strains the eyes over time.

## Four variants, one cohesive system

**Dark Modern** ‚Äî Unified dark background with an ergonomic slate-blue foundation. Every panel shares the same tone, creating a focused, distraction-free environment. Best for low-light or mixed-light environments.

**Dark Contrast** ‚Äî A deeper editor sits inside a slightly lighter panel shell, giving the code area its own architectural weight. Good for those who like visual structure between editor and sidebar.

**Light Soft** ‚Äî A warm cream-white background that replaces the harsh pure white of most light themes. Easy on the eyes in bright offices, near windows, or on high-brightness displays.

**Light Contrast** ‚Äî Split light backgrounds with clear depth between editor and UI panels. Maximum legibility for daytime use or paper-like aesthetics.

## Syntax palette

The syntax colors share the same cool-warm balance across all variants:

- **Keywords** ‚Äî soft violet
- **Functions** ‚Äî slate blue
- **Strings** ‚Äî sage green
- **Variables** ‚Äî warm amber
- **Numbers** ‚Äî dusty rose
- **Comments** ‚Äî muted, stays out of the way

## Installation

**One-click** ‚Äî opens directly in VS Code:

[![Install in VS Code](https://img.shields.io/badge/Install%20in-VS%20Code-0078d7?logo=visualstudiocode)](vscode:extension/equinox-team.equinox-colors)

**Extensions panel** (`Ctrl+Shift+X` / `‚åò‚áßX`) ‚Äî search **Equinox Colors**

**Quick Open** (`Ctrl+P` / `‚åòP`):

```
ext install equinox-team.equinox-colors
```

After installing, open **Command Palette** (`Ctrl+Shift+P` / `‚åò‚áßP`) ‚Üí **Preferences: Color Theme** ‚Üí select the Equinox variant you want.
